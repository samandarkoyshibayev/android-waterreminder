A health and fitness mobile application designed to help users maintain optimal hydration levels. This project focuses on background scheduling, local data persistence, and a modern, reactive user interface.

ech Stack & Architecture
Built with a strong emphasis on performance and offline-first capabilities using the latest Android development tools.

Architecture: Clean Architecture for a robust separation of business logic and UI.

Pattern: MVI (Model-View-Intent) to ensure a predictable and unidirectional data flow.

UI Framework: Jetpack Compose with Material 3, providing a modern and fluid user experience.

Local Database: Room Database for persistent storage of daily water intake and historical data.

Background Tasks: WorkManager for reliable scheduling of hydration reminders.

Notifications: Notifications API for timely user alerts.

Dependency Injection: Dagger Hilt for modular and testable code.

Concurrency: Kotlin Coroutines & Flow for reactive data streams from the database to the UI.

Key Features
Custom Goal Setting: Allows users to set and adjust daily water intake targets.

Automated Reminders: Smart notification system powered by WorkManager that works even when the app is closed.

Progress Visualization: Detailed statistics by day and week, featuring interactive charts to track hydration trends.

Intake History: A comprehensive log of consumption history managed through Room.

Modern Material 3 UI: Dynamic color support and polished components following the latest Android design guidelines.
